## Profile

自律機械知能 (Autonomous Machine Intellgence, AMI) の研究開発を行っています。機械学習、自律 AI、ロボティクス領域が専門です。

また、プログラミング言語は [**Python**](https://www.python.org) に熟達しており、自律機械知能の研究に関連した堅牢かつ複雑なシステム・ライブラリの開発も行っています。

2024 年 4 月より [**ジー・オー・ピー株式会社 / GOP co., ltd.**](https://www.gop.co.jp)に所属。業務を行いつつ、弊社のスポンサーのもと研究開発を行っています。

### 自律機械知能 "P-AMI\<Q>" の研究・開発

好奇心ベースの自律機械知能であり、[**VRChat**](https://hello.vrchat.com) に実装してその振る舞いを観察しています。彼らはとても興味深い存在であり、それを生み出し、探求するのがたまらなく楽しいのです。

P-AMI\<Q>の研究・開発には ML 集会や様々な場所で出会った多くの方々にご協力をいただいております。誠にありがとうございます。

## Developments

**P-AMI\<Q>**

- [**pamiq-core**](https://github.com/MLShukai/pamiq-core): 推論と学習をリアルタイムで非同期に並行して行うためのコアライブラリです。
- [**pamiq-io**](https://github.com/MLShukai/pamiq-io): 音声や映像、マウス、キーボードといった機械知能エージェントのための入出力機能をまとめたライブラリです。
- [**pamiq-recorder**](https://github.com/MLShukai/pamiq-recorder): 自律機械知能の入出力を記録するためのライブラリです。
- [**PynkTromboneGym**](https://github.com/Geson-anko/PynkTromboneGym): 人間の発生器官シミュレータ`PinkTrombone`を強化学習環境のように扱えるようにしたライブラリです。(現在このライブラリ自体はメンテナンスしていませんが、今後機械知能の発声器官を造る際に参考にする可能性があります。)

**ROS2**

- [**ROS2 NDArray Messages**](https://github.com/Geson-anko/numpy_ndarray_msgs): [ROS2](https://www.ros.org)上で Numpy などの多次元配列を送受信するためのメッセージライブラリです。Python には[`ros2-ndarray-msg-utils`](https://pypi.org/project/ros2-ndarray-msg-utils/)もあります。
- [**ROS2 UV Template**](https://github.com/Geson-anko/ros2_uv_template): 高速なパッケージマネージャ[**uv**](https://docs.astral.sh/uv/)を用いた ROS2 のテンプレートプロジェクトです。[解説ブログはこちら](https://qiita.com/GesonAnko/items/510eeade1f8ada302b9b)

**Python**

- [**Python UV Template**](https://github.com/Geson-anko/python-uv-template): 私が用いている Python のテンプレートプロジェクトです。pytest, coverage, 性的型検査, コードフォーマット, 自動テスト, 開発コンテナセットアップ, とにかく私が普段使うものは揃っています。
